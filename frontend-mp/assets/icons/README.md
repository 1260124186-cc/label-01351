## 图标规范

### 尺寸要求
- **推荐尺寸**: 81 × 81 像素
- **最小尺寸**: 40 × 40 像素
- **最大尺寸**: 不超过 200 × 200 像素

### 格式要求
- **文件格式**: PNG（支持透明背景）
- **文件大小**: 单个图标不超过 40KB

### 颜色规范
- **未选中状态**: #999999（灰色）
- **选中状态**: #8B4513（乡土棕，主题色）

### 设计建议
- 使用简洁的线性图标风格
- 保持图标视觉重量一致
- 图标内容居中，四周留有适当边距
- 建议使用单色图标，便于状态切换

## 图标设计参考

```
首页 (home)     - 房屋/小屋图标
人物志 (people) - 人群/双人物图标
投稿 (publish)  - 加号/编辑/笔图标  
我的 (mine)     - 用户/人物图标
```

## 获取图标

可从以下渠道获取免费图标：
- [阿里巴巴矢量图标库 (iconfont)](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [IconPark](https://iconpark.oceanengine.com/)

## 添加图标后

将 6 个图标文件放入本目录后，小程序即可正常显示 tabBar 图标。
